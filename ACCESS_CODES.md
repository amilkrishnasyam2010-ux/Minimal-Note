# Access Codes Reference

This document provides a complete reference for access codes used in the Minimal Notes application.

## How Access Codes Work

Access codes are generated using the formula:
```
'MN' + base64(fileKey).substring(0, 5).replace('=', '')
```

Where `fileKey` is the resource identifier (e.g., `Maths_1`, `Physics_2_QB`, `Chemistry_3_OW`).

## Free Resources (No Access Code Required)

The following resources are available without an access code:
- Maths Chapter 6 - Question Bank (`Maths_6_QB`)
- Maths Chapter 7 - Question Bank (`Maths_7_QB`)
- Physics Chapter 5 - Question Bank (`Physics_5_QB`)
- History Chapter 4 - Question Bank (`History_4_QB`)
